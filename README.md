# ⌨️ ローマ字タイピング

子ども向けのローマ字タイピング練習Webアプリ。**ホームポジション（指1本）から文章まで**、やさしい順にステップアップして、成長が見えるように作っています。題材は子どもが大好きな **マインクラフト**。

- 公開URL: https://mikushiba.github.io/romaji-typing/
- リポジトリ: https://github.com/mikushiba/romaji-typing
- 単一HTML＋共有モジュール / 依存ゼロ（`file://` でもGitHub Pagesでも動く）
- 姉妹アプリ：[漢字クイズ](https://github.com/mikushiba/kanji-quiz)・[さんすうクイズ](https://github.com/mikushiba/sansu-quiz)（UI/UX・育成思想を踏襲）

---

## 超初心者むけの工夫

- **キーボード図ガイド**：次に押すキーをハイライト。指ごとに色分け（左こゆび〜右こゆび）。F/J に出っぱり表示。
- **おてほん（両手の指）**：両手10本の指をキーボードに重ねて表示。ホーム（A S D F／J K L ;／親指=スペース）に置かれ、**次に押す指だけがホームから対象キーまで移動**して届く。指は太さ・長さがそれぞれ違い、指先＋つめ＋手のひらで「手」に見える。
  - **ヒントが出たとき（数秒つまったとき）だけ**表示。すらすら打てているときは出ない。
- **考えこむとヒント**：約 **2.5秒**（`HINT_DELAY`）さわらないと、押すキーがピカピカ＋「○○ゆびで！」のゆびガイド＋ヒントバナーが出る。
- **ローマ字ヒント**：かな（大きく）の下に、おすすめのローマ字つづりを表示。
- 上記3つ（キーボード／おてほん／ローマ字ヒント）は**トップ画面の「せってい」**でトグルON/OFFできる（保存され全ゲーム共通。プレイ画面に切りかえUIは無い）。

## ゲーム（むずかしさ順）

トップは「ゲームを えらぼう」。上から やさしい順に並び、各カードに難易度バッジ（信号色）が付く。

| # | ゲーム | むずかしさ | 内容 |
|---|--------|-----------|------|
| 0 | ホームポジション | 🟢 やさしい | `f j d k s l a ;` |
| 1 | あいうえお 1もじ | 🟢 やさしい | あ から ん まで **1もじずつ・順番どおり** |
| 2 | あいうえお ブロック | 🟢 やさしい | あいうえお・かきくけこ … を **5もじずつ・順番どおり** |
| 3 | かんたんなことば | 🟠 ふつう | マイクラの みじかい ことば |
| 4 | とくべつルール | 🟠 ふつう | っ・ん・ー・ゃゅょ |
| 5 | マイクラフレーズ | 🟠 ふつう | マイクラの みじかい フレーズ |
| 6 | みじかい文 | 🔴 むずかしい | マイクラの みじかい 文 |
| 7 | 文章 | 🔴 むずかしい | 、。のある ながい 文章 |
| — | じゆうにうつ | 🟣 じゆう | **お手本なしの自由うちこみ**（別ページ `free/`）。下記参照 |

- 難易度の区切り：`index.html` のステージ生成部で `i<=2=やさしい / i<=5=ふつう / それ以上=むずかしい`。
- **やさしいの3つ（1もじ・ブロック含む）は `ordered:true`**：シャッフルせず順番どおり・全問出題。それ以外はシャッフルして一部出題。

### じゆうにうつ（自由うちこみ・`free/index.html`）
お手本のかなを打つ既存ゲームとは別の、**自由記述モード**。
- ローマ字で打つと **その場でひらがなに変換**して表示（簡易IME。`Romaji.toKana()`）。`Backspace`＝けす、`Enter`＝改行、スペース可。
- 白紙がこわくないよう、**ゆるいお題ヒント**（`lessons.js` の `free.prompts`：すきな たべもの／ともだちの なまえ 等）を出し、「べつの お題」で切替。
- 「**できた！**」で、確定したかな数を **称号（打ったかな累計）・かな図鑑・あそんだ回数** に反映（`recordKana`/`bumpChars`/`notePlay`）。お手本との照合や採点（はやさ・せいかくさ）は無し。
- 変換は本アプリのローマ字ルールに準拠：`し`=shi/`ち`=chi/`ふ`=fu のみ、`ん`=nn（`si`/`ti`/`hu` は確定しない）。

### レベル解放（鍵）について
- 各ゲームは前のゲームをクリアすると解放される設計（`unlock` / `need`）。
- **現在はテスト用に全解放中**。`index.html` の `var UNLOCK_ALL = true;` を **`false` にすると鍵が復活**（やさしい順に解放）。

## 成長実感のしくみ

- **結果画面**：はやさ・せいかくさ%・**前回比/最高記録**・打ったかな を表示。
- **にがてキー分析**：キーごとのミスを記録 → 結果で「にがてキー」を提示。
- **育成（全ゲーム合算・localStorage）**
  - 🐣 **称号**：タイピング 10きゅう 〜 マスター（打ったかな累計で昇格）。トップにバーで表示。
  - 📚 **かな図鑑**：打てたかなを集める（5かいで ★マスター）。
  - 🏅 **メダル**：16種（あそんだ回数・連続・ノーミス・はやさ・図鑑 など）。
  - 🔥 **コンボ** / 💾 **セーブ**。
- トップのサマリーは4つ：**打ったかな・さいこうの はやさ・れんぞく・あそんだ回数**。
- **かな図鑑・メダル・せっていは常時表示せず、ボタンを押すとモーダルで開く**（背景クリック・×・Esc で閉じる）。
- **せってい（トップ画面）**：タイピング画面の表示（キーボード／ゆびのおてほん／ローマ字ヒント）を3つのトグルで ON/OFF。`store.settings` に保存し全ゲーム共通で反映。タイピング画面側に切りかえUIは持たない（設定を読むだけ）。
- **きろくの ほぞん／よみこみ**（トップ下部）：進捗（localStorage の `store` 全体）を JSON ファイルに書き出し／読み込み。別端末への移行やバックアップに使う。読み込み時は確認のうえ上書きしてリロード。漢字クイズと同じ方式（`exportSave`/`importSave`）。

> 「はやさ」＝ 1分あたりの打鍵数（旧称KPM）。子ども向けに表記を「はやさ」へ統一。

### 見た目
- 算数・漢字アプリのように **カラフル**（サマリー／ゲームカードを虹色に色分け）。
- **飾りの絵文字は使わない**方針（タイトル・見出し・カードのアイコン等は無し）。
  - 例外：**メダル一覧（モーダル内）のアイコンのみ絵文字**（漢字/算数アプリと同じ見せ方）。

## 構成

```
index.html            … トップ＝称号バー＋サマリー＋ゲーム選択（カラフル・難易度バッジ）
                         ＋「かな図鑑をみる」「メダルをみる」「せってい」ボタン→モーダル
                         ＋「きろくを ほぞん／よみこむ」（JSONで進捗を入出力）
play/index.html       … タイピング本体（キーボード図・おてほん・ヒント・結果。表示ON/OFFはトップのせっていに従う）
free/index.html       … じゆうにうつ（自由うちこみ。ローマ字→ひらがな変換・お題ヒント）
shared/typing-ui.css  … 共有デザイン（キーボード図・指の色分け・おてほん・モーダル）
shared/typing-core.js … ローマ字判定エンジン Romaji（判定＋toKana逆変換）＋ 育成エンジン TypingCore
shared/lessons.js     … ゲーム教材（かな↔ローマ字・マイクラ単語/文章）
shared/build.txt      … 公開中の版番号（キャッシュ自動更新用）
validate.mjs          … 全問が打ち切れるか検証（CIで自動実行）
deploy.sh             … commit+push＋共有更新時のキャッシュ版数そろえ
```
- localStorageキー：`typingQuiz.v1`（漢字 `kanjiQuiz.v1`・算数 `sansuQuiz.v1` とは別）。

### ローマ字判定（`shared/typing-core.js` の `Romaji`）
かなを「トークン」に分け、各トークンに複数の正しいつづりを許可（`し`=shi、`つ`=tsu/tu、`ち`=chi、`じゃ`=ja/jya/zya …）。`っ`は次の子音をかさねる、`ん`は基本 `nn`（子音の前では `n` も可）、`ー`は `-`。カタカナは自動でひらがなへ正規化。`;` などの記号キーも入力可。1文字打つごとに「正しいつづりの途中か」を判定する。

### キャッシュ自動アップデート
各HTML先頭で `shared/build.txt` を取得し、自分の `BUILD` 定数より新しければ1回だけ取り直す（開いているタブ＝家族のタブレットにも更新が届く）。`deploy.sh` が `build.txt`・各HTMLの `?v=` と `BUILD` を同じ版にそろえる。

## 開発・公開

```bash
node validate.mjs                 # 教材の検証（全問が打ち切れるか）
./deploy.sh "メッセージ"           # コミット＆プッシュ（Pagesへ反映）
```
`shared/` を変えると `deploy.sh` がキャッシュ版を上げるので、開いているタブにも自動で更新が届く。CI（GitHub Actions）でも push 時に `validate.mjs` が走る。

## 変更履歴（主な調整）

- 初版：ホームポジション〜文章の8ゲーム、キーボード図ガイド、ヒント、育成（図鑑/称号/メダル）、マイクラ題材。
- `;` が入力できないバグ修正（入力許可リストに `;` 追加）。
- テスト用に全ゲーム解放（`UNLOCK_ALL` フラグ）。
- 「あいうえお」を**順番練習**に再構成：1もじ（あ→ん）／ブロック（あいうえお→わをん）を `ordered` で順番出題。
- **おてほん（両手の指オーバーレイ）**を追加 → 指らしい見た目（太さ・長さ・つめ・手のひら）→ **ヒント時のみ表示**に変更 → トグルON時はヒント状態をリセット。
- トップを整理：ずかんリンク・説明文・各種カウント・リセットボタンを削除、ラベルを日本語化。
- トップを**カラフル**化（サマリー／ゲームカードを色分け）、ゲーム説明を簡潔化。
- かな図鑑・メダルを**ボタン→モーダル**表示に変更。
- レベルカードの番号とタイトルの重なりを解消（番号を上の行へ）。
- 「レベル」→「**ゲームを えらぼう**」、Lv番号→**むずかしさ（やさしい/ふつう/むずかしい）バッジ**に変更。「クリア」表記を削除。
- サマリーから かな図鑑・メダルの数を削除。
- **「KPM」表記を全廃 →「はやさ」に統一**（チップ・結果・メダル名）。
- **飾りの絵文字を全削除**（タイトルアイコン・連続の炎・結果の祝い絵文字 など）。メダル一覧のアイコンのみ残置。
- **ローマ字判定を厳格化**：`ち`=chi のみ（ti廃止）・`ふ`=fu のみ（hu廃止）・`し`=shi のみ（si廃止）。
- **きろくの ほぞん／よみこみ**を追加（トップ下部。進捗をJSONで入出力、別端末への移行用。漢字クイズと同方式）。
- **表示設定をトップ画面の「せってい」へ集約**：プレイ画面のチェックボックスを廃止し、3項目をトグル化＋保存（`store.settings`）。プレイ画面は設定を読むだけ。
- **「じゆうにうつ」（自由うちこみ）を追加**：お手本なしで好きな言葉を打つと ひらがなに変換（`Romaji.toKana`）。お題ヒント＋できた！で育成（称号・図鑑）に反映。
