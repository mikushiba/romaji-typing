# ⌨️ ローマ字タイピング

子ども向けのローマ字タイピング練習Webアプリ。**指1本（ホームポジション）から文章まで**、段階を上るごとに成長が見えるように作っています。題材は子どもが大好きな **マインクラフト**。

- 公開URL: https://mikushiba.github.io/romaji-typing/
- 単一HTML＋共有モジュール / 依存ゼロ（`file://` でもGitHub Pagesでも動く）
- 姉妹アプリ：[漢字クイズ](https://github.com/mikushiba/kanji-quiz) ・ [さんすうクイズ](https://github.com/mikushiba/sansu-quiz)（UI/UX・育成思想を踏襲）

## 超初心者むけの工夫
- **キーボード図ガイド**：次に押すキーをハイライト。指ごとに色分け（左こゆび〜右こゆび）。
- **考えこむとヒント**：約2.5秒さわらないと、押すキーがピカピカ＆「○○ゆびで！」と教えてくれる。
- **ローマ字ヒント**：かな（大きく）の下に、おすすめのローマ字つづりを表示（ON/OFF可）。
- **やさしい→文章**：母音→五十音→単語→特別ルール→フレーズ→短文→文章 と無理なく進む。

## 成長カリキュラム（Lv0→Lv7）
| Lv | ステージ | 内容 |
|----|---------|------|
| 0 | ⌨️ ホームポジション | `f j d k s l a ;` |
| 1 | 🅰️ ぼいん | あいうえお |
| 2 | 🔤 ごじゅうおん | か行〜わ行・ん |
| 3 | 🐱 かんたんなことば | つち・いし・ベッド…（マイクラ） |
| 4 | ✨ とくべつルール | っ・ん・ー・ゃゅょ（クリーパー等） |
| 5 | 💬 マイクラフレーズ | つるはしでほる 等 |
| 6 | 📝 みじかい文 | よるになるとモンスターがでてくる 等 |
| 7 | 📖 文章 | 「、」「。」のある長め文章 |

各レベルは前レベルを **クリアすると解放**。

## 成長実感のしくみ
- **結果画面**：KPM（はやさ）・せいかくさ％・**前回比/最高記録**を毎回表示。
- **にがてキー分析**：キーごとのミスを記録 → 結果で「にがてキー」を提示。
- **育成（全レベル合算）**：📚かな図鑑（打てたかなを集める・5回で★）／🐣称号（10きゅう〜マスター）／🏅メダル16種／🔥コンボ／💾セーブ（localStorage）。

## 構成
```
index.html            … トップ＝レベル選択＋育成サマリー（称号・図鑑・メダル）
play/index.html       … タイピング本体（キーボード図・ヒント・結果）
shared/typing-ui.css  … 共有デザイン（キーボード図・指の色分け）
shared/typing-core.js … ローマ字判定エンジン Romaji ＋ 育成エンジン TypingCore
shared/lessons.js     … レッスン教材（かな↔ローマ字・マイクラ単語/文章）
shared/build.txt      … 公開中の版番号（キャッシュ自動更新用）
validate.mjs          … 全問が打ち切れるか検証（CIで自動実行）
deploy.sh             … commit+push＋共有更新時のキャッシュ版数そろえ
```
- localStorageキー：`typingQuiz.v1`（漢字 `kanjiQuiz.v1`・算数 `sansuQuiz.v1` とは別）。

## ローマ字判定
かなを「トークン」に分け、各トークンに複数の正しいつづりを許可（`し`=shi/si、`つ`=tsu/tu、`ち`=chi/ti、`じゃ`=ja/jya/zya …）。`っ`は次の子音をかさねる、`ん`は基本 `nn`（子音の前では `n` も可）、`ー`は `-`。1文字打つごとに「正しいつづりの途中か」を判定します。

## 開発・公開
```bash
node validate.mjs                 # 教材の検証
./deploy.sh "メッセージ"           # コミット＆プッシュ（Pagesへ反映）
```
shared/ を変えたら deploy.sh が `build.txt`・各HTMLの `?v=` と `BUILD` を同じ版にそろえるので、開いているタブにも更新が届きます。
